# FluxDL Browser Extension

## What it does

- Adds **Send to FluxDL** in the right-click context menu.
- Sends media/page URLs to your running FluxDL app using `?extUrl=`.
- Includes popup button to send the current tab.

## Install (Chrome/Edge)

1. Open extensions page (`chrome://extensions` or `edge://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder: `browser-extension/fluxdl-send`.

## Configure

- Open extension **Options**.
- Set your FluxDL app URL (default: `http://localhost:3000`).

## Usage

- Right-click a video/link/audio/page and choose **Send to FluxDL**.
- FluxDL opens and pre-fills the URL in the Download tab.
